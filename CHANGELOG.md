## [0.1.0] - 2025-11-23

### Added
- Initial release of toyyibpay-ruby gem
- Support for toyyibPay API v1
- Bank operations (list banks, FPX banks)
- Category management (create, retrieve)
- Bill operations (create, retrieve, get transactions)
- Package operations
- Settlement operations
- User operations
- Rails integration with generators and helpers
- Sandbox mode support
- Comprehensive error handling
- Configuration management
- Thread-safe client implementation
- Full test suite
- Documentation and examples
